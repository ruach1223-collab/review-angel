# 리뷰천사 마케팅 플랜

## 현실 인식

- 유료 고객: 0명
- 마케팅 예산: 거의 없음 (1인 운영)
- 강점: 워크위드 10년 CS 운영 경험 + 상담사풀 + 제품 완성도 높음
- 약점: 아직 아무도 모름

**원칙: 돈이 없으면 시간을 쓴다. 시간도 없으면 기존 자산을 쓴다.**

---

## Phase 1: 이번 주 (0원, 시간만)

### 1-1. 워크위드 기존 고객 전환 (Day 1)

가장 빠르고 확실한 방법. 이미 신뢰 관계가 있는 사람.

- 워크위드 기존 고객 중 이커머스 셀러 있는지 확인
- 있으면: "사장님, 저희가 리뷰 자동 답변 서비스도 시작했어요. 무료로 한번 써보시겠어요?"
- 없으면: 기존 고객에게 "혹시 주변에 쿠팡이나 스마트스토어 하시는 분 계세요?" 소개 요청

### 1-2. 셀러 커뮤니티 가입 + 관찰 (Day 1~2)

바로 광고하면 강퇴당함. 먼저 들어가서 구경하고, 사람들이 뭘 힘들어하는지 파악.

**가입할 곳 (우선순위):**

| 순위 | 커뮤니티 | URL | 회원수 | 이유 |
|:---:|------|-----|------|------|
| 1 | 아이보스 | i-boss.co.kr | 39만 | 국내 최대, 셀러+마케터 |
| 2 | 셀러오션 | sellerocean.com | - | 셀러 전용 커뮤니티 |
| 3 | 네이버 카페 "쿠팡 셀러 모임" | 카페 검색 | 수만 | 쿠팡 셀러 집중 |
| 4 | 카카오 오픈채팅 | "쿠팡 셀러" 검색 | 수천 | 실시간 소통 |

**들어가서 할 일:**
- "리뷰 답변", "CS 직원", "악성 리뷰" 키워드로 검색
- 이런 고민 글을 쓴 사람이 진짜 타겟
- 2~3일은 답변만 달아주기 (광고 X)

### 1-3. 유용한 글 1개 올리기 (Day 3~4)

**제목 예시:** "쿠팡 리뷰 답변, 이렇게 쓰면 별점 올라갑니다 (실전 템플릿 공유)"

**내용:**
- 리뷰 유형별 답변 템플릿 (긍정/중립/부정)
- 부정 리뷰 대응 시 하면 안 되는 것 3가지
- 응답률이 검색 순위에 미치는 영향
- 마지막에 살짝: "이런 게 귀찮으시면 자동화 도구도 있어요 → review1004.co.kr"

**이게 왜 효과적인지:**
- 진짜 도움이 되는 글이라 사람들이 읽고 저장함
- 댓글로 자연스러운 대화 시작
- "이거 만든 사람이에요?" → "네, 제가 만들었어요" → 신뢰

### 1-4. 리뷰 성적표 활용 (Day 4~5)

**아이보스/카페에 올릴 글:**
"내 스토어 리뷰 관리 점수, 무료로 확인해보세요"
→ review1004.co.kr/scorecard 링크

- 이메일 수집됨
- 점수 낮은 분에게 "리뷰천사로 이 부분 개선할 수 있어요" 팔로업

## Phase 2: 2~4주차 (0원~소액)

### 2-1. 네이버 블로그 SEO (무료)

셀러들이 검색하는 키워드로 블로그 글 작성:

| 키워드 | 검색 의도 | 글 제목 예시 |
|--------|----------|------------|
| "쿠팡 리뷰 답변 방법" | 직접 하려는 사람 | "쿠팡 리뷰 답변 잘 쓰는 법 + 자동화 팁" |
| "CS 직원 채용 비용" | 직원 고민 중인 사장님 | "CS 직원 1명 실제 비용 계산해봤습니다 (월 300만원)" |
| "악성 리뷰 대응" | 스트레스 받고 있는 사람 | "악성 리뷰 달렸을 때 전화하면 70% 수정됩니다" |
| "쿠팡 판매자 점수 올리기" | 순위 올리고 싶은 사람 | "리뷰 응답률 100% 유지하는 법" |
| "이커머스 CS 외주" | CS 외주 찾는 사람 | "CS 외주 비용 비교: 대행사 vs AI vs 리뷰천사" |

**글 구조:**
1. 문제 공감 (혼자 하느라 힘들죠)
2. 해결법 제시 (직접 할 수 있는 팁 3가지)
3. 더 편한 방법 소개 (리뷰천사)
4. 무료 체험 링크

**주 2개 글 목표.** 3개월이면 24개 글이 쌓이고, 검색 유입이 시작됨.

### 2-2. 쿠팡에서 타겟 셀러 직접 찾기 (무료)

**방법:**
1. 쿠팡에서 카테고리별 인기 상품 검색
2. 리뷰 50개+ 있는데 답변이 거의 없는 스토어 찾기
3. 그 스토어 이름으로 네이버/카카오에서 사장님 연락처 찾기
4. DM: "사장님 스토어 리뷰 147개 중 답변 8개뿐이에요. 응답률 5%인데, 업종 평균이 35%거든요. 리뷰천사가 자동으로 답변해드릴 수 있어요."

**이게 왜 강력한지:** 구체적 숫자로 들어가니까 "아 내 스토어 얘기네" 싶어서 무시가 안 됨.

### 2-3. 카카오톡 채널 개설 (무료)

- 리뷰천사 카카오톡 비즈니스 채널 개설
- 1:1 상담, 무료 체험 안내, 성적표 결과 발송 용도
- 랜딩에 카카오 채널 추가 버튼 넣기

## Phase 3: 2~3개월차 (소액 예산)

### 3-1. 셀러 교육 파트너십

**가장 효과적인 파트너:**

| 파트너 | 접근법 | 비용 |
|--------|--------|------|
| 라이프해킹스쿨 (쿠팡 부트캠프) | "수강생에게 리뷰천사 3개월 무료 제공" 제안 | 무료 (서비스 제공) |
| 셀러대학 | CS/리뷰 관리 커리큘럼에 추천 도구로 등록 | 무료~소액 |
| 단아쌤TV (유튜버) | 리뷰 답변 자동화 영상에 협찬 | 50~100만원 |

**왜 교육 파트너십인가:** 셀러 교육 들으러 온 사람 = 성장하려는 셀러 = 돈 쓸 의향 있는 사람. 타겟 정확도가 높음.

### 3-2. 아이보스 광고 (월 30~50만원)

- 아이보스 커뮤니티 내 배너 광고 또는 스폰서 게시글
- 39만 마케터/셀러에게 직접 노출
- "CS 직원 채용 월 300만원, 리뷰천사는 월 9,900원부터" 카피

### 3-3. 카페24/아임웹 앱스토어 입점 (무료)

- 카페24 앱스토어에 리뷰천사 등록
- 자사몰 운영 셀러가 앱스토어에서 CS 도구 검색 시 노출
- 크리마, 브이리뷰가 이미 입점해 있는 형태 참고

## 퍼널 설계

```
[인지] 블로그/카페 글, 커뮤니티 답변
    ↓
[관심] 리뷰 성적표 무료 진단 (이메일 수집)
    ↓
[체험] 14일 무료 체험 (회원가입)
    ↓
[전환] 체험 7일차 팔로업 → 12일차 유료 전환 유도
    ↓
[유지] 주간 성과 리포트 → 업그레이드 제안
```

**핵심 지표:**
- 성적표 진단 수 → 주 10건 목표
- 무료 체험 전환율 → 20% 목표 (성적표 → 체험)
- 유료 전환율 → 30% 목표 (체험 → 결제)
- 월 신규 유료 고객 → 첫 달 2명, 둘째 달 5명

## 주간 루틴 (1인 기준, 주 5시간)

| 요일 | 할 일 | 시간 |
|------|------|:---:|
| 월 | 네이버 블로그 글 1개 작성 | 1시간 |
| 화 | 아이보스/카페 답변 달기 (3~5개) | 30분 |
| 수 | 쿠팡에서 타겟 셀러 3명 찾기 + DM | 1시간 |
| 목 | 성적표 진단 결과 팔로업 이메일 | 30분 |
| 금 | 체험 중인 셀러 팔로업 + 피드백 수집 | 1시간 |
| 토~일 | 카카오 오픈채팅 참여 (자연스럽게) | 1시간 |

**총 주 5시간.** 이 정도면 혼자서도 가능해요.

## 하지 말아야 할 것

1. **네이버/구글 광고부터 시작하기** — 유료 고객 0명인데 광고비 태우면 돈만 날림
2. **한번에 많은 채널 운영** — 블로그+카페+인스타+유튜브 다 하면 아무것도 안 됨. 블로그+카페 2개만 집중
3. **"AI" 강조하기** — 셀러는 AI에 관심 없음. "내 CS를 대신 해준다"에 관심 있음
4. **완벽한 마케팅 자료 만들고 시작하기** — 지금 당장 카페 가입하고 글 1개 올리는 게 낫
